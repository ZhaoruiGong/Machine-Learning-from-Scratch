# Machine-Learning-from-Scratch
## Introduction
This repository documents my journey of learning machine learning and deep learning by implementing core algorithms and models from scratch. The goal of this repo is not to build a production-ready library, but to better understand how things work under the hood — through coding, experimenting, debugging, and rewriting ideas in my own way.

## What’s inside

This repository includes implementations and notes for topics such as:
Linear Regression with Stochastic Gradient Descent
Penalized Linear Regression
k-Nearest Neighbors and K-Means Clustering
Decision Trees and Random Forests
Principal Component Analysis with Kernels
Deep Neural Networks
......
and is being updated continuously.

## A note on references

This repository is a study/learning repo, and many implementations are inspired by or adapted from educational resources, tutorials, books, and online videos.

In particular, I have learned a lot from:

YouTube machine learning / deep learning tutorials
Dive into Deep Learning by Aston Zhang, Zachary C. Lipton, Mu Li, and Alexander J. Smola
other open educational materials, blog posts, and public code examples

I do not claim that any implementation here is fully original.
Many parts were written while following or adapting ideas from existing learning resources.
